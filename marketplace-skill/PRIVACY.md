# Privacy Policy

**@cronocode/react-box Claude Code Plugin**

Last updated: 2026-03-28

## Data Collection

This plugin **does not collect, store, transmit, or process any personal data or usage information**. It consists entirely of static instructional content (SKILL.md) that is read locally by Claude Code.

## What This Plugin Does

This plugin provides AI assistant guidance for using the @cronocode/react-box library. It contains only markdown text with coding instructions, examples, and best practices. It does not:

- Collect or transmit any data
- Access the network or external services
- Store any information about users or their projects
- Use cookies, analytics, or telemetry
- Process or access any personal information

## Third-Party Services

This plugin does not integrate with or send data to any third-party services.
